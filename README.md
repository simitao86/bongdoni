# 봉도니 랜딩페이지

감귤밭 옆, 제주 봉개동 현지인 고깃집 **봉도니**의 모바일 우선 통합 랜딩(허브) 페이지.
Tmap·인스타·구글 등 모든 채널의 "홈페이지 URL" 칸에 넣는 단일 페이지입니다.

- 단일 `index.html` — 외부 라이브러리·프레임워크 0, CSS·JS 인라인
- 시스템 폰트만 사용 → 빠르고 가벼움, 외부 요청 0
- 모바일 우선 반응형 (데스크톱에선 가운데 정렬), 고정 하단 바(전화·예약·길찾기)

---

## ✅ 교체할 항목 체크리스트

대부분 `index.html` 맨 아래 `<script>` 안의 **`PLACE` 설정 한 곳**에서 끝납니다.

### 1) 길찾기 좌표 — 가장 중요 ⭐
`PLACE.lat`, `PLACE.lng` 에 봉도니 정확한 위도·경도 입력.
- **얻는 법(30초):** [카카오맵](https://map.kakao.com) → `제주 번영로 589` 검색 → 정확한 위치에서 **마우스 우클릭 → "좌표 복사"** → 위도=lat / 경도=lng 붙여넣기.
- 비워두면 버튼이 **주소·이름 검색**으로 동작해요(목적지는 정확, 턴바이턴은 아님). 좌표를 넣으면 정확한 길안내로 자동 업그레이드됩니다.
- 좌표 순서 주의: 카카오=`(위도,경도)`, Tmap=`goalx=경도/goaly=위도` (코드에 이미 맞춰져 있음).

### 2) 링크 (`PLACE` 설정)
| 항목 | 설명 |
|---|---|
| `naverBooking` | 네이버 예약 URL. **비우면** 자동으로 네이버 플레이스로 연결 |
| `kakaoChannel` | `http://pf.kakao.com/_xxxxx`. **비우면** 카카오 채널 버튼 자동 숨김 |
| `instagram` | 인스타 주소. **비우면** 푸터 아이콘 자동 숨김 |
| `naverPlace` | 이미 입력됨(플레이스 1228480297) |
| `tel` | 이미 입력됨(0507-1398-6240) |

### 3) 사진·영상·지도 (`<!-- 교체: ... -->` 주석 위치)
- **히어로 배경**: 기본은 감성 그라데이션. 사진/영상으로 바꾸려면 ① `.hero-media` 주석 해제 + src 교체, ② `<section class="hero">` → `class="hero photo"`(어두운 스크림+흰 글씨 자동). 파일은 같은 폴더에 `hero.jpg` 또는 `hero.mp4`(+`poster.jpg`).
- **갤러리 사진 6장**: 각 `<img src="...">`를 실제 사진으로 교체(`loading="lazy"` 유지).
- **지도**: `.map-ph` 자리에 정적 지도 이미지(`<img loading="lazy">`) 또는 네이버/카카오 지도 iframe.
- **공유 미리보기**: `og.jpg`(1200×630) 추가 후 `<head>`의 og:image 주석 해제.

> ⚠️ 콘텐츠 정확성: 가게는 **"고깃집"**으로 표기(흑돼지집 ❌), 흑돼지는 **가브리살·항정살 특수부위로만**, "제주산" 정확 표기, 최상급("최고/1등") 표현 금지.

## 🚀 정적 호스팅에 올리는 법

### A. 로컬 미리보기
`index.html` 더블클릭 → 브라우저에서 바로 열림.

### B. Netlify Drop (가장 쉬움 · 무료)
1. https://app.netlify.com/drop 접속
2. `bongdoni` 폴더를 통째로 드래그 & 드롭
3. 즉시 `https://이름.netlify.app` 공개 URL 생성

### C. GitHub Pages (무료)
1. GitHub에 새 repo 생성 → `index.html` 루트에 업로드
2. Settings → Pages → Source: *Deploy from a branch* → `main` / `(root)` → Save
3. 1~2분 뒤 `https://아이디.github.io/repo이름/` 공개
4. (원하시면 Tria 허브 때처럼 repo 생성·push·배포를 대신 처리해 드릴 수 있어요.)

배포 후 그 URL을 **Tmap·인스타·구글 비즈니스 등 "홈페이지" 칸**에 넣으면 끝입니다.
